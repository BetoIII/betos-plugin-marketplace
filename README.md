# Bulk Messenger Desktop

**Bulk messaging plugin for Claude Desktop** - Compose and send personalized SMS messages to multiple contacts using templated with variables for personlization.


## Usage

You have to add this marketplace before you can install plugins.

1. Add the Marketplace to you claude code instance **/plugin markplace add BetoIII/betos-plugin-marketplace**
2. Install the plugin **/plugin install bulk-messenger@BetoIII/betos-plugin-marketplace**
3. Activate the plugin with the **/send-messages** slash command
